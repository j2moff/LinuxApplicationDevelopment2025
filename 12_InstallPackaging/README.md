# Romanizer
<!-- you're a romanizer, baby -->
Программа романизации текстов на русской кириллице.

## Сборка
Для сборки программы выполните следующие команды:
```sh
autoreconf --force --install --symlink
./configure --prefix=LOC
make
```

`LOC` - каталог установки программы (например, `/usr/`).

## Установка
Для установки программы после её сборки выполните следующую команду:
```sh
make install
```

## Использование
```
romanizer [INFILE OUTFILE]
```

Аргументы:
- INFILE  - входной файл с текстом на кириллице.
- OUTFILE - выходной файл с текстом на латинице.

При отсутствии INFILE и OUTFILE будут использоваться STDIN и STDOUT соответственно.

Для просмотра manpage выполните следующую команду:
```sh
man romanizer
```

Для просмотра HTTP-документации по адресу `localhost:8000` выполните следующую команду:
```sh
make http
```

Для запуска unit-тестов выполните следующую команду:
```sh
make check
```

Для удаления программы выполните следующую команду:
```sh
make uninstall
```
